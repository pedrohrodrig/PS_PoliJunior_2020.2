Como funciona?

Você pode:
- Preencher manualmente a tabela abaixo com os dados para que o sistema faça o cálculo da nota caso não queira cadastrar a matéria
- Selecionar dentre as matérias já carregadas no sistema ou cadastrar outras para que elas sejam incluídas na lista

O botão "Limpar" faz com que a tabela fique em branco novamente.
